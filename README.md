# Web Demo - Professional Nail Products Website

Website demo bán hàng mỹ phẩm nail chuyên nghiệp với thiết kế hiện đại và đẹp mắt.

## Tính năng

- **Header & Navigation**: Header sticky với logo, menu điều hướng, và các icon tiện ích
- **Hero Sections**: 
  - Hero section 1: Giới thiệu gel polish HEMA-Free & TPO-Free với carousel
  - Hero section 2: "Why You'll Love Us" với thông điệp thương hiệu
- **Product Categories**: 5 danh mục sản phẩm chính (Acrylic Powder, Gel Nail Polish, Dip Powder, Gelly Tips, Nail Art)
- **Celebrity Endorsements**: Grid hiển thị các người nổi tiếng yêu thích sản phẩm
- **Testimonials**: Carousel hiển thị đánh giá từ khách hàng
- **Footer**: Footer đầy đủ với newsletter subscription, links, và payment methods

## Công nghệ sử dụng

- HTML5
- CSS3 (với Grid và Flexbox)
- JavaScript (Vanilla JS)
- Font Awesome Icons

## Cách sử dụng

1. Mở file `index.html` trong trình duyệt web
2. Website đã được thiết kế responsive, hoạt động tốt trên desktop, tablet và mobile

## Tính năng tương tác

- **Carousel**: Click vào arrows hoặc dots để chuyển slide
- **Testimonial Carousel**: Navigate giữa các testimonials
- **Scroll to Top**: Button xuất hiện khi scroll xuống
- **Email Subscription**: Nhập email để đăng ký nhận tin
- **Smooth Scrolling**: Navigation links scroll mượt mà
- **Hover Effects**: Các element có hiệu ứng khi hover

## Responsive Design

Website được thiết kế responsive với breakpoints:
- Desktop: > 1024px
- Tablet: 768px - 1024px
- Mobile: < 768px

## Màu sắc chính

- **Nâu đậm**: #3d2817 (Text chính)
- **Đỏ đậm**: #8b2635 (Accent, buttons)
- **Nâu nhạt**: #b8866f, #f5e6d3 (Background)
- **Hồng nhạt**: #f5d7d7 (Accent elements)

## Cấu trúc files

```
├── index.html      # File HTML chính
├── styles.css      # File CSS styling
├── script.js       # File JavaScript cho interactivity
└── README.md       # File documentation
```

## Tùy chỉnh

Bạn có thể dễ dàng tùy chỉnh:
- Màu sắc trong file `styles.css`
- Nội dung text trong file `index.html`
- Tính năng tương tác trong file `script.js`
